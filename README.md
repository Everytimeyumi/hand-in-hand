## hand-in-hand
 技术栈主要是：taro+typescript+vue3+pinia,目前主要是着重小程序端的展示，腾讯ligth公益比赛入围项目，暂时开放前端部分，通过这个项目也可以帮助你快速使用Taro开发一个属于你自己的小程序。

### 快速开始

```
  第一步
  首先需要在 src 目录下创建一个config.ts, 
  第二步
  yarn install
  or
  cnpm install
  or
  npm install
  第三步
  yarn dev:weapp
  or
  cnpm dev:weapp
  or
  npm dev:weapp

  
```
